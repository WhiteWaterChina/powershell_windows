1. 此工具是使用powershell编写的windows的网线拔插监测工具
2. 执行方法为使用管理员权限开启powershell，然后cd到脚本所在目录，执行即可。
3. 执行时会提示输入一个要测试的网口可以ping通的IP。然后会显示当前所有物理网卡的信息，输入y/Y来确认，或者n/N来结束测试。
4. 测试开始前，请保持要测试的网口的网线是插入状态。其他不测试的网口的状态无所谓，插不插网线都行。测试过程中不要拔插其他网口。只测试一个网口。
5. 测试结果保存在以开始是的时间戳生成的文件夹中。测试过程中的详细记录保存在plug.log文件中，详细记录了网口信息及每次测试结果。
6. 一次只能测试一个网口。测试完成一个之后使用ctrl+c结束测试。
